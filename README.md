Pastebin in Node.js and Express 3
=================================

This is a free software pastebin developed in Node.js and Express 3.

Features
========

* client-side encryption
* syntax-highlighting (through Syntaxhighlighter)
* REST API
* pasting through a raw TCP socket
* more
